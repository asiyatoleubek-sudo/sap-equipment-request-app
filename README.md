# Equipment Request App (SAP Build)

## Overview
This project is a business application built in SAP Build to manage equipment requests.
Employees can submit requests and managers can approve or reject them.

## Features
- Equipment request creation
- Approval workflow
- Comments and tracking
- Centralized request storage
- Status tracking

## Tools & Technologies
SAP Build

## What I Learned
- Building business apps in SAP Build
- Workflow configuration
- UI design for enterprise apps
